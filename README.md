# ReactiveHttp

一个基于 Kotlin + Jetpack + Coroutines + Retrofit 封装的网络请求框架

## 导入依赖

```groovy
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

	dependencies {
		implementation 'com.github.leavesC:ReactiveHttp:1.0.0'
	}
```
